# FPGA-PONG-GAME
An accelerometer-controlled breakout game implemented in Verilog on the Digilent Nexys A7 FPGA board. Tilt the board to move the paddle and bounce the ball across. Built with a 6-state FSM, SPI-driven ADXL362 accelerometer input, real-time VGA rendering, 7-segment score display, and progressive difficulty.
